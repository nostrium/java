Nostr compatibility
------------------

NOSTR means Notes and Other Stuff Transmitted by Relays.
It is a protocol published in 2020, documented through
the [100 NIP](https://github.com/nostr-protocol/nips).

NIP stands for NOSTR Implementation Possibilities, more
details can be found at the [Wikipedia](https://en.wikipedia.org/wiki/Nostr)
page.

This platform uses NOSTR as basis for the user accounts,
it is also preparing to support each of the NIP described
on the list below so that it can function as a fully
operational notes relay.
